# Personal Growth OS

A modular, mobile-first task management and personal tracking system designed to help you grow.

## Phase 1: Task Engine
* **Features:** Add/Edit tasks, priority levels, recurrence logic (daily/weekly), and browser notifications.
* **Tech Stack:** React, TypeScript, Replit, LocalStorage.

## Future Roadmap
* **Phase 2:** Nutrition/Diet Module (Macros & Weight tracking)
* **Phase 3:** Workout Tracker (Sets/Reps/History)
